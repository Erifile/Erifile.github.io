Основные компоненты системного блока, их назначение

Материнская плата

Материнская плата - "фундамент", на который крепятся большинство остальных компонентов. Она содержит множество разъёмов для них, объединяя компьютер в единое целое.

Поэтому материнскую плату всегда выбирают в связке с другими компонентами, удостовериваясь в их совместимости. Например, "сокет" на мат. плате - разъём для процессора, должен ему подходить.

Процессор (ЦП, CPU)

Основное назначение процессора — это обработка машинного кода, из которого состоят все программы.

    Основные функции процессора:
  • Выполнение инструкций. Программы отправляют процессору машинные инструкции в виде двоичного кода, которые он интерпретирует и выполняет на аппаратном уровне.
  • Управление памятью. ЦП обращается к оперативной памяти, записывает информацию в постоянную память или берёт данные из собственной кэш-памяти.
  • Обработка данных. Вместе с программными инструкциями поступают данные, которые надо обработать и вернуть результат.
  • Управление системой. CPU связывает между собой работу жёстких дисков, видеокарты, оперативной памяти и других компонентов.

    • Оперативная память (ОЗУ, RAM)

    Назначение оперативной памяти - хранение и выделение памяти для задач, работающих на ПК в текущий момент. ОЗУ также следует выбирать в связке с предыдущими компонентами: у ОЗУ есть модули (DIMM..), у которого в свою очередь должен быть свой тип, совпадающий с мат. платой (DDR3, DDR4..). Всё взаимосвязано, поэтому нужно ориентироваться и на процессор - слишком низкая мощность ОЗУ не раскроет потенциал мощности ЦП.
